## About Laravel QRcode Frontend

Laravel QRcode Frontend is the second phase of the Laravel QRcode Processor Platform.

Laravel QRcode Frontend is a basic e-commerce platform that demonstrates the use of API and QRcodes.
- The calling and consuming of API.
- Getting tokens.
- Generate and display QRcode
